Cryptml3

How do you use this?

  1. Enter an offset, between -9007199254740991 and 9007199254740991.
  2. Type your message.
  3. Click ENCRYPT.
  4. Send the message to a friend.
  5. If they have the same offset, they can decrypt your message with the "DECRYPT" setting, all on this very same site.

Note: you can layer many offsets and encryptions to make things more impossible to intercept.

Note 2: this does not support charCodes below 32 or above 126. Newline characters, for instance, get corrupted.
